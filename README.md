# FitForge Gym App

FitForge is a modern fitness website built with Next.js. It includes a home page, BMI and fitness calculator, exercises by muscle group, progress tracker, nutrition corner, and blog content.

## Features

- Modern landing page for a fitness brand
- BMI and fitness calculator
- Exercise library grouped by muscle group
- Simple weekly progress tracker
- Nutrition tips and healthy recipe ideas
- Blog section with fitness motivation and recovery advice

## Tech Stack

- Next.js
- React
- CSS

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Run the development server
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser

## Project Structure

- app/ - main Next.js pages and layout
- package.json - project scripts and dependencies

## Deployment

You can deploy this project easily on Vercel or any other platform that supports Next.js.
